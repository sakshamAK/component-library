Avatars

An avatar is the face of user's profile. It is the user profile picture. It is a small image consisting of user's face generally to give user's content a visual identity.

Profile Avatar

Profile Avatars are the primary avatars that will be placed by default. They can be accessed by the avatar class.

Examples

Avatar Sizes

There are multiple sizes for avatars. These can be accessed by class names sm, md, and lg.

Examples

Square Avatar

Avatars in square shape can be accessed by the class name 'square'
