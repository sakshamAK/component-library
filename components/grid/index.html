Grid

A grid is a layout style that helps you arrange your content in a particular order

2x2 Grid

To access grid styles use .grid-box .rc-2 class names in your container to access 2x2 column grid.

Examples

block(1, 1)
block(1, 2)
block(2, 1)
block(2, 2)
