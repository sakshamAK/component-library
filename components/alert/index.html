Alerts

An alert is a short message that only pops up when some information is needed to be pass on to a user on a certain action.

Default Alerts

Default alerts are simple alerts with background colors that indicates their purpose.

Examples

Primary! This is a Primary Alert - Check it now!
Secondary! This is a Secondary Alert - Check it now!
Success! This is a Success Alert - Check it now!
Danger! This is a Danger Alert - Check it now!
Warning! This is a Warning Alert - Check it now!
